## README: WaterBalance_ClimateVsLULC ##

# There are three parallel sets of analysis:
-PheasantBranch = this analyzes historical streamflow data from the Pheasant Branch USGS gauging station (1974-2016)
-PheasantBranch-AgroIBIS = this analyzes AgroIBIS output for the Pheasant Branch Subwatershed only (1986-2013)
-Yahara-AgroIBIS = this analyzes AgroIBIS output for the entire Yahara Watershed for a factorial set of 16 scenarios (1986-2070)
